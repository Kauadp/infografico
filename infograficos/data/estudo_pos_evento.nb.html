peak_demanda <- as.data.frame.matrix(table(dados$data_emissao, dados$hora_emissao)) |>
  rownames_to_column(var = "data_emissao") |>
  pivot_longer(
    cols = -data_emissao,
    names_to = "hora_emissao",
    values_to = "contagem"
  ) |>
  mutate(
    hora_emissao = as.factor(hora_emissao),
    data_emissao = ymd(data_emissao)
  )

Vamos gerar o heatmap de faturamento


dados |>
  mutate(
    dia_da_semana = wday(data_emissao, label = TRUE, abbr = FALSE),
    
    hora_emissao = as.factor(hora_emissao)
  ) |>
  group_by(dia_da_semana, hora_emissao) |>
  summarise(
    receita_bruta = sum(Valor.total, na.rm = TRUE),
    .groups = 'drop'
  ) |>
  ggplot(aes(x = hora_emissao, y = dia_da_semana, fill = receita_bruta)) +
  geom_tile(color = "white", linewidth = 0.5) +
  scale_fill_viridis_c(
    name = "Receita Bruta (R$)",
    labels = scales::label_number(prefix = "R$")
  ) +
  labs(
    title = "Heatmap: Receita Bruta por Hora do Dia e Dia da Semana",
    x = "Hora da Emisso",
    y = "Dia da Semana"
  ) +
  scale_y_discrete(limits = rev) + # Mantm a inverso para que a leitura seja de cima para baixo
  theme_minimal()

  • O maior faturamento bruto ocorre no Sbado, especificamente das 12h s 18h, sendo o pico mximo concentrado por volta das 16h.

  • Existe uma clara separao entre as mtricas. O maior Ticket Mdio ocorre no incio do dia (10h/11h), enquanto o maior Volume de Transaes ocorre no final do dia (19h).

  • O dia da semana o fator mais influente no faturamento, com o sbado e o domingo apresentando faturamento significativamente maior que os dias teis.

  • Os horrios da manh (10h-15h) de quarta, quinta e sexta-feira representam um ponto de baixa receita bruta, confirmando que o alto ticket mdio no compensa o baixssimo volume de transaes nessas janelas.

Concluso

  • O pico de Receita Bruta (Valor Total) ocorre no Sbado, das 12h s 18h, com concentrao mxima por volta das 16h. Este o momento ideal para alocar recursos e pessoal, pois a receita por hora atinge o mximo.

  • Existe uma clara e estatisticamente significativa contradio no comportamento de compra:

    • Maior Ticket Mdio: Ocorre nas horas da manh (10h/11h), indicando que, embora o volume seja baixo, os clientes nessas horas gastam significativamente mais por transao.

    • Maior Ticket Mdio: Ocorre nas horas da manh (10h/11h), indicando que, embora o volume seja baixo, os clientes nessas horas gastam significativamente mais por transao.

  • O Dia da Semana o fator mais forte para determinar o Valor Faturado, com o Sbado e o Domingo impulsionando a maior parte da receita bruta semanal.

  • O dia 2025-10-21 (incio da semana) o ponto de menor Receita Bruta Total, mas possui o maior Ticket Mdio por transao.
